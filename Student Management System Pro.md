**Project Name**

**Student Management System Pro**



**Project Scope**



The Student Management System Pro is a web-based application developed using HTML, CSS, and JavaScript. The project allows users to manage student records efficiently through a simple and interactive interface.



&#x20;**Objectives**



\* Add student details.

\* Display student information in a tabular format.

\* Calculate average CGPA automatically.

\* Display total number of students.

\* Identify the top-performing student.

\* Delete student records when required.



&#x20;**Technologies Used**



\* HTML5

\* CSS3

\* JavaScript



&#x20;**Features**



1\. Student Registration

2\. Student Record Management

3\. Automatic CGPA Calculation

4\. Top Performer Identification

5\. Delete Student Functionality

6\. Responsive User Interface



&#x20;**Working Procedure**



1\. Enter student details in the input fields.

2\. Click the "Add Student" button.

3\. Student information is displayed in the table.

4\. Dashboard cards update automatically.

5\. Students can be removed using the Delete button.
